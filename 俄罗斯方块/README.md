# 俄罗斯方块游戏 - 黑白版

一个经典的俄罗斯方块游戏，采用黑白配色设计，简洁美观。

## 游戏特性

- 🎮 经典俄罗斯方块玩法
- ⚫⚪ 黑白配色设计
- 📱 响应式界面设计
- 🎯 分数系统和等级提升
- 💾 本地最高分记录
- ⏸️ 暂停/继续功能
- 🔄 下一个方块预览

## 游戏控制

- **←→** 左右移动方块
- **↓** 快速下降
- **↑** 旋转方块
- **空格** 暂停/继续游戏

## 游戏规则

1. 控制下落的方块，使其填满水平行
2. 当一行被完全填满时，该行会消失，获得分数
3. 随着消除行数增加，游戏等级提升，方块下落速度加快
4. 当方块堆积到顶部时，游戏结束

## 如何开始

1. 在浏览器中打开 `tetris.html` 文件
2. 点击"开始游戏"按钮
3. 使用键盘控制方块移动和旋转
4. 享受游戏！

## 技术特点

- 使用 HTML5 Canvas 进行游戏渲染
- 纯 JavaScript 实现，无需额外依赖
- 响应式设计，支持不同屏幕尺寸
- 本地存储保存最高分记录
